# ActivityHub

开源的**活动筹办系统**，基于 [Koa][1]、[TypeScript][2] 和 [LeanCloud][3] 的 **Node.js RESTful API** 项目。

[![NPM Dependency](https://david-dm.org/kaiyuanshe/ActivityHub.svg)][4]

## 主要特性

1. [LeanCloud 手机短信验证码登录](source/controller/Session.ts)

## 本地开发

```shell
npm install

lean login
lean switch
lean up
```
